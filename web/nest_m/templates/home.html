Suggested Reading & References

Throughout your introduction to Python, I'll add links here to compelling outside material. If you come across articles or links that you find particularly helpful, please share and we can add it to the list for the rest of the class. :)

  • The Python Standard Library By Example, by Doug Hellmann

    (Highly recommended) One of Python's greatest features is the standard library, the core set of modules available to every Python application. Through the examples in this book, you can see practical real world code snippets that can guide you as you learn to interact with modules and extensions in Python.

  • 10 Myths of Enterprise Python, Paypal

    An interesting read from Paypal addressing many of the commonly asked questions about the Python language.

  • The Hitchhiker's Guide to Python

    One of Python's strongest features, and the developer community at large, is the abundance of community maintained documentation. You can learn any language just be reading the docs! For Python, one of the best beginner documentation resources is The Hitchhiker's Guide to Python; diving into great detail about many of the "Best Practices" in Python. It can be a little complicated at first, however it makes for a great reference tool to go back to as you learn more about programming.

  • The Hitchhiker's Guide to Python -- Structuring Your Project

    One of the most valuable sections of the Hitchhiker's Guide to Python is explaining Python's module, package and import architecture. As your projects grow you will begin to have an unruly amount of code. This will be very exciting but also difficult to manage, so you may want to organize your code into related functional areas. Using modules and pacakges allow you to split up your application to keep your project organized, as well as share code within the application.

  • The Hitchhiker's Guide to Python -- Virtual Environments

    The "Getting Started with Python" slides cover how to use pip to install PyPi libraries to add extended functionality to Python. However, different Python applications may rely on different libraries, or even different versions of the same library, which can cause conflicts on your computer. Covered in the Hitchhiker's Guide to Python is the solution to how to install Python libraries in a special environment so they're specific to the project you are working on. This tool is called a Virtual Environment.
