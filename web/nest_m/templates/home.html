Suggested Reading & References

Throughout your introduction to Python, I'll add links here to compelling outside material. If you come across articles or links that you find particularly helpful, please share and we can add it to the list for the rest of the class. :)

  • The Python Standard Library By Example, by Doug Hellmann

    (Highly recommended) One of Python's greatest features is the standard library, the core set of modules available to every Python application. Through the examples in this book, you can see practical real world code snippets that can guide you as you learn to interact with modules and extensions in Python.

  • 10 Myths of Enterprise Python, Paypal

    An interesting read from Paypal addressing many of the commonly asked questions about the Python language.
